What is Simplex

Simplex is a modern web content management and publishing application. With Simplex you can create, curate and publish content online easily and customize everything just the way you want it.

Why Simplex

Ease of effective use is the main goal of Simplex. Not every project benefits from web content simplification, but there are many that do - and Simplex aims to meet those with such needs.

Strong Foundation

Simplex is built on strong foundations with great tools that are available across the app for you to use. Based on proven technology including Symfony2 components, Doctrine, Twig, Silex and others, Simplex is not just easy to use, but also to extend and customize. It uses Composer for dependency management and is developed with best practices in mind.

Getting Started

To get started with Simplex you can try the demo online, browse the source code or learn how to download, install and play with it.

Features - Core Version

Content features

  • Intuitive content management and publishing
  • Powerful content publishing options
  • Transparent support for multimedia content
  • Automatic content enrichment and classification
  • Per-post keywords and metadata specification

Form features

  • Support for user defined web forms
  • Forms get posted to database
  • Form results available in admin panel
  • Simple Twig form helper included

Page features

  • Structure defined with user-defined SEO-friendly URLs
  • Page Queries define dynamic data to display on pages
  • Static content can be included in pages directly
  • Page-specific metadata supported for every page

Media features

  • Drag and drop media import
  • Automatic media thumbnail generation
  • Automatic media metadata extraction
  • Automatic image watermarking if enabled
  • User defined image sampling quality

Templating and theming

  • Supports templates written in pure PHP and Twig
  • Templates receive data as defined in pages
  • Creating your own templates is as easy as HTML
  • Assetic is available for theme asset management

Users and security

  • User groups for editors and administrators
  • Full user profile for each user
  • Support for public user registration
  • Secure password and credentials storage
  • Account activation and reactivation via email
